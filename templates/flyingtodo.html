About

This project was made with one word in mind: Simplicity. From my own experience, I have a lot of problems working without a guideline. Without my ToDo List, I'm getting lost in my own work, forgetting or drifting from one task to another.
This resolves everything : A simple program, without a hundred features that distracts its users, you can tick boxes and add tags, that's all !

Challenges

This app was a big challenge for me: this is my first ever electron app. Not knowing how to use it was the main problem, but also learning to setup the app on the PC side was a challenge. I also struggled with tags and code formatting, as the first version's code was really messy.

Solutions

Well, there aren't that many solutions: I had to read. I read the documentation of Electron , which, I am lucky, is pretty straight-forward! It was also a great occasion to use the windows terminal, using commands like `cd /d` or `npm run`. I had to install and use NPM, which taught me about versionning and git tags ! For the formatting of code, I had to think and learn about transferring a forEach item to an external function, allowing me to create reusable functions! The biggest struggle was with tags: it wouldn't work, but I finally found a solution using forEach and creating small divs inside an item, with an ID.

Future

In the future, I want to continue this app. This was made because I have an objective: I want to code every app of my PC myself, so that I won't need other external applications. I want FlyingToDo to be a sort of "productivity hub", like notion, but way more simple, without paywalls and without a thousand features that distracts its users.
