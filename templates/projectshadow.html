About

This game was my first contact with code and creation. I wanted to create a small game with a character, enemies and bonuses. This was made to learn Unity, C# and game-development.

Challenges

The biggest challenge was that I had no experience with coding nor with Unity. This created problems as I was watching tutorials on Unity and how to create elements, how to make them move, etc.

Solutions

As said, I watched some tutorials which gave me something to start with. I was using tutorials to get the idea, and then tried to replicate the features alone and understanding the code. but as Unity was a bit too hard for a complete beginer, I stopped developing it. I look forward to continuing it one day because game development is one of the subjects that interest me in development!
